Shared Services Platform

Built on Google Compute Engine, complete with Apache web server and mod_WSGI, a PostgreSQL database and a Django web application for managing users and the worker apps. It serves as a deployment platform for software tools used to provide services to both internal and external clients.

Travel Crawler

A series of web crawlers that help the client gather and centralize the data they need for their day-to-day activity. The applications use Firefox in headless mode and are deployed on Ubuntu, on both AWS EC2 and Google Compute Engine.
Data Related Services, Process Automation
Salesforce Match Control

An automated process that takes information from the client's web applications, runs a series of data cleaning, manipulation and matching procedures to make up for missing data, and verifies that information against data in Salesforce. It then packs everything in a report that is concomitently sent by email to the final user and saved to a database. Helped reduce user workload from days to a couple of hours.

Desktop application that applies machine learning techniques on historical data, in order to reduce the time necessary for the realization of critical report from two days to about thirty minutes.

Field Report Centralizer

Realized in Excel VBA, takes as input a folder containing non-tabular reports, compiled by the members of the field force. The data gets collected, structured and then used to generate a centralized, tabular report, used as a starting point for further analysis.

Data Set Deduplicator

Implementation of various string matching techniques, that help identify and remove duplicate rows in large manually submitted data sets. It follows a set of rules provided by the user but also take into acount that the duplicate rows might not be perfectly identical because of the errors that often occur during manual typing.
