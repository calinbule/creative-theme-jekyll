Salesforce Validator
Salesforce Validator

An automated process that downloads data, in the form of CSV files, from the client's web applications, runs a series of cleaning and matching procedures and verifies that information against the Salesforce database for validation. It then packs everything in a report that is concomitantly sent by email to the final user and saved to a database. It helped reduce user workload from a few days to a couple of hours.

Data Set Deduplicator
Data Set Deduplicator

Implementation of various string matching techniques, that help identify and remove duplicate rows in large manually submitted data sets. It follows a set of rules provided by the user but also take into account that the duplicate rows might not be perfectly identical because of the errors that often occur during manual typing.
